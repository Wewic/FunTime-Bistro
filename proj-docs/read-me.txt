----[Running PHP files locally]----
1. Terminal -> folder containing PHP files
2. Type: php -S localhost:8000
3. Type into browser: localhost:8000/[FILE-NAME].php

----[Starting MySQL Server locally]----
1. Start up MySQL server in terminal/system preferences [OSX]/application
2. Terminal command: mysql -u root -p
